## How to Use

1. Print these QR codes
2. Open the KrishiTrace app
3. Go to **Scan** tab
4. Scan any QR code
5. View the full traceability details

## Scanning Shows

When scanned, the app displays:
- ✅ Blockchain Verified badge
- 🌾 Crop: [crop type] & Quantity
- 👨‍🌾 Farmer Information: Name, farm size, method
- 📖 Farmer Story: Background about the farmer
- 📍 Location & Harvest Date
- 💰 Pricing info
- 🌡️ IoT sensor data (temp, humidity)
